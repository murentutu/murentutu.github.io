Docker+Calibre-web搭建个人图书馆

使用docker搭建个人图书馆

安装docker

  1. 更新yum

  2. 安装需要的软件包

  3. 设置yum源

  4. 查看所有仓库中的docker版本，选择特定版本安装。

  5. 启动并加入开机启动

  6. 验证安装是否成功

  7. 安装配置文件，输入 vim /etc/sysctl.conf 打开配置文件，Calibre-Web 需要开启IPv4转发，CentOS默认不支持IPv4转发。

  8. 激活配置文件

拉取calibre镜像

  1. 使用technosoft2000/calibre-web镜像

  2. 启动calibre-web

  3. 要启动calibre-web，有一个关键的依赖是metadata.db文件，没有这个文件calibre-web无法正常运行。打开本地电脑calibre客户端软件，查看书库的目录。

  4. 到对应路径下，找到metadata.db文件，将其上传到/data/caliber/books目录下。

  5. 为了数据库配置能够正常写入，我们需要给/data/calibre 目录设置为可写入的权限

    chmod -R 777 /data/calibre/*

  6. 登录calibre-web，浏览器访问ip地址，默认的用户名为admin，密码为admin123

  7. 配置书库地址为/books,然后点击保存

  8. 点击admin，修改语言为中文，记得在最下面点击save进行保存

  9. 上传书籍和在线阅读

    点击管理权限->编辑基本配置->基本配置->启动上传->保存

  10. 配置启动上传后，就可以将我们本地的电子书上传到服务器中

  11. 选择本地的一本电子书

  12. calibre-web会自动提取电子书中的元数据

  13. 之后的操作和客户端calibre类似，就不细说了。

最大子序和-Leetcode

post @ 2021-11-04

题目：给定一个整数数组 nums ，找到一个具有最大和的连续子数组（子数组最少包含一个元素），返回其最大和。

分析

方法一：贪心算法

class Solution:
    def maxSubArray(self, nums: List[int]) -> int:
        # 贪心
        # 如果当前指针所指元素之前的和小于0，则丢弃当前元素之前的数列
        # 复杂度：空间复杂度O(1),时间复杂度O(n)
        cur_sum = max_sum = nums[0]
        for i in range(1, len(nums)):
            cur_sum = max(nums[i],cur_sum+nums[i])
            max_sum = max(cur_sum,max_sum)
        return max_sum

方法二：动态规划

class Solution:
    def maxSubArray(self, nums: List[int]) -> int:
        # 动态规划
        # 若前一个元素大于0，则将其加到当前元素上
        # 复杂度：空间复杂度O(1),时间复杂度O(n)
        n = len(nums)
        for i in range(1,n):
            if nums[i-1] > 0:
                nums[i] += nums[i-1]
        return max(nums)

Java Set.add()方法

post @ 2021-11-04
  • Java 集合类中的 Set.add() 方法用来向 Set 集合添加对象。
  • 如果 Set 集合中已经包含相同的对象，则不改变 Set 集合。
  • 该方法返回值为 boolean 对象，如果 Set 集合中不包含要添加的对象，则添加对象并返回 true，否则返回 false。

用法:向Set集合添加对象（可用作哈希表查找，如果哈希表中已经有该对象，则不再添加）

public static void main(String[] args){
    Set set = new HashSet();  //定义Set集合对象
    set.add(new Date());  //向集合中添加日期对象
    set.add("apple");  //向集合中添加对象
    set.add("book");
    set.add("apple");
    set.add(new Integer(3));
    int size = set.size();  //获取集合的大小
    System.out.println("Set集合的大小为:"+size);  //将集合大小输出
}

Leetcode217：Contains Duplicate

Given an integer array nums, return true if any value appears at least twice in the array, and return false if every element is distinct.

给定一个整数数组，判断是否存在重复元素。

如果存在一值在数组中出现至少两次，函数返回 true 。如果数组中每个元素都不相同，则返回 false 。

代码解答

class Solution {
    public boolean containsDuplicate(int[] nums) {
        Set<Integer> set = new HashSet<Integer>();
        for(int x : nums){
            if(!set.add(x)){
                return true;
            }
        }
        return false;
    }
}

1. 概述

​ 感知机是二类分类的线性分类模型，其输入为实例的特征向量，输出为实例的类别，取+1和-1二值。感知机学习旨在求出将训练数据进行线性划分的分离超平面，为此，导入基于误分类的损失函数，利用梯度下降法对损失函数进行极小化，求得感知机模型。它是神经网络与支持向量机的基础。

什么是超平面？

 可以理解为对于一个n维空间，超平面就是一个n-1维的子空间，可以用来分割n维空间。
