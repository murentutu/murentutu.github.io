官方文档上找到http.client

HTTPConnection对象

HTTPConnetcion代表和HTTP的一个连接，它的实例化需要传入一个主机和可选的端口号。如果没有传入端口号，首先会尝试从传入的主机字符串中提取出端口号，没有则使用默认的端口号。

conn = http.client.HTTPConnection('news.hut.edu.cn')    # 参数主机不需要填入http://

实例化以后，拥有以下方法：

  • 参数method表示使用的请求方法，通常是POST或者GET
  • url就是地址栏中删去主机以后的那一部分
  • body默认为none，如果给定值，它的数据会在请求头之后发送。
  • headers是随请求发送的额外HTTP请求头的映射。

结合后面的response方法，可以写一个简单的爬取hut新闻页面的代码，如下所示：

import http.client
# class http.client.HTTPConnection(host, port=None, [timeout, ]source_address=None, blocksize=8192)
conn = http.client.HTTPConnection('news.hut.edu.cn')
print(conn)
# 同时发送多个GET，需要采用异步，多线程等方式
# HTTPConnection.request(method, url, body=None, headers={}, *, encode_chunked=False)
conn.request('GET',"/info/1034/7484.htm")
resp = conn.getresponse()
# print(resp.status)
datas = resp.read()
# 根据网页编码方式对应解码
datas = datas.decode("utf-8")
print(datas)
conn.close()

想要将爬取的页面单独成一个txt文件，使用下面的代码

with open('news.txt','w',encoding="utf-8") as file:
    file.write(datas)

成功爬取到hut新闻网的源网页

学习爬虫第三步——获取需要的数据（采用正则表达式匹配需要的内容）

​ 正则表达式是一个牛逼哄哄的东西，对我来说，它是一个可以匹配字符串，并且可以提取对应字符串的一种语法规则。听起来很复杂，学起来也感觉很复杂。至于用起来，需要一点点摸索。

​ 对于hut新闻网，我们需要获取每个新闻的标题以及对应的超链接，日期等。查看新闻网的源代码中，发现都有这样的特点

根据这个格式我们可以写出这样的正则表达式来进行匹配，（）里面的内容是我们想要提取出来的。

# <a href="http://news.hut.edu.cn/info/1002/7441.htm">省委第三巡视组向湖南工业大学党委反馈巡视情况  </a>
# <em>[11-30] </em>
res = re.findall("<a href=\"(.+?)\">[\s]*(.*)[\s]*</a>[\s]*?<em>([\s\S]*?)</em>",datas,re.M)

然后需要将成功匹配到的内容写入到一个文本中，方便查看。

wr_content = ''
for i in res:
    wr_content = f'{wr_content}{i[1]}\n{i[0]}\n{i[2]}\n\n'    # 格式化输出
with open('NewsTitle.txt','w',encoding="utf-8") as file:
    file.write(wr_content)

import http.client
import re
'''
    遗留问题；1.新闻网站通常有多页数据，如何获取？
            2.如何将我们获取的数据添加到数据库？
'''
# 第一步 创建HTTPConnection对象
# class http.client.HTTPConnection(host, port=None, [timeout, ]source_address=None, blocksize=8192)
conn = http.client.HTTPConnection('news.hut.edu.cn')
print(conn)
# 第二步 使用request()方法向爬取网站发送http请求
# HTTPConnection.request(method, url, body=None, headers={}, *, encode_chunked=False)
conn.request('GET',"/rdyw.htm")     # 同时发送多个GET，可以采用异步，多线程等
# 第三步 使用getresponse()方法获取返回的响应内容
resp = conn.getresponse()
datas = resp.read()
# 第四步 将获取到的源代码进行解码
datas = datas.decode("utf-8")       # 根据网页编码方式对应解码
# 第五步 将源代码写入news.txt
with open('news.txt','w',encoding="utf-8") as file:
    file.write(datas)
# 第六步 利用正则表达式匹配我们需要的内容：新闻标题、对应的链接、日期
# <a href="http://news.hut.edu.cn/info/1002/7441.htm">省委第三巡视组向湖南工业大学党委反馈巡视情况  </a>
# <em>[11-30] </em>
res = re.findall("<a href=\"(.+?)\">[\s]*(.*)[\s]*</a>[\s]*?<em> \[([\s\S]*?)\]</em>",datas,re.M)
# 第七步 将匹配到的内容写入NewsTitle.txt中
wr_content = ''
for i in res:
    wr_content = f'{wr_content}{i[1]}\n{i[0]}\n{i[2]}\n\n'    # 格式化输出
with open('NewsTitle.txt','w',encoding="utf-8") as file:
    file.write(wr_content)
conn.close()
